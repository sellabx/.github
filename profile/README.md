---

## What We Build

- **Backend Infrastructure** — Scalable services, APIs, and data pipelines designed for high-throughput environments
- **Developer Tooling** — CLI tools, SDKs, and automation that reduce friction in day-to-day engineering workflows
- **Platform Utilities** — Cross-cutting libraries for observability, configuration, and deployment management

## Principles

- **Correctness first** — We ship software that works, not software that seems to work
- **Minimal surface area** — Simple APIs, clear contracts, no unnecessary abstractions
- **Open by default** — Governance, contribution standards, and decision-making are documented and public
